離散時間システム

観測データが、離散時間システムで支配されていると仮定した時の解釈について述べる

Note

実際は、観測データを支配しているシステムの定義は非常に困難

\(x(t+1) = x(t) + F(x(t))\)
\(F\): 生産率と死亡率を合わせた関数（\(F = B(x(t)) - D(x(t))\)）
