カテゴリ変数を用いた回帰1

カテゴリ変数を用いた回帰では、以下を行なっている.
- 回帰係数は基準カテゴリとの差
- 「あるカテゴリが基準カテゴリと平均的に差があるか」を検定

\(y = x_{p1}b_1 + x_{p2}b_2 + b_0\)
上式の内、\(x_{p2}\)が2つの要素(AとB)を持つカテゴリ変数とした時、カテゴリ変数をダミー変数にして,
\(A = 1\)
\(B = 0\)

ここで、それぞれの要素について考えた時、以下のようになる.
Aの時 \(y = x_{p1}b_1 + b_2 + b_0\)
Bの時 \(y = x_{p1}b_1 + b_0\)
つまり、同じ\(x_{p1}\)の値を持っている時は、平均的に\(b_2\)だけ差があるのを意味する.
